# TPfinal
## 📂 Estructura del Proyecto
En cada modulo se configuran también las rutinas de atencion 
```text
TPfinal/
└── config.c/
    └── (Configuraciones del UART timer y change notification)
└── routines.c/
    └── (rutinas de atención del timer y change notification) 
└── comunicacion.c/
    └── (rutinas de atencion de la UART y protocolos de recepción y envios)
└── main.c/
    └── (declaracion de variables, structs y logica general del programa)
```
