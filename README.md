# TPfinal
## 📂 Estructura del Proyecto
En cada modulo se configuran también las rutinas de atencion 
```text
TPfinal/
└── ADC.C
    └── (Configuraciones del adc y dma)
└── TIMER.c/
    └── (Configuraciones del timer)
└── UART.c/
    └── (Configuraciones de la UART) 
└── main.c/
    └── (Configuración de puertos, modulos e inicio del programa)
```
